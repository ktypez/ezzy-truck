# Web Design Skill

## Overview

This skill assists in designing and building visually appealing, maintainable, and accessible web applications. It provides guidance on modern UI patterns, effective CSS architecture, and core accessibility principles.

## Core Guidance

- **UI/UX Patterns:** For component design and layout strategies, refer to [UI Patterns](references/ui-patterns.md).
- **CSS Architecture:** For structuring CSS, theme management, and performance, refer to [CSS Architecture](references/css-architecture.md).

## Usage

Use this skill when:
- Designing new components or layouts.
- Refactoring CSS for better maintainability or performance.
- Adding accessibility features to existing components.
- Setting up consistent design systems (colors, spacing, typography).
